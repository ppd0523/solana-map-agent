# Solana Map - Agent
*Solana Map Agent* calls the request of *Solana Map Server* instead and returns the response.
 

## Quick start
```bash
#/bin/bash
$ ./agent 
```

## Help
```bash
$ ./agent --help
```

## Requirement
```bash
# linux
apt install libzmq3

# osx
brew install zmq
```


## Environment
* Unix
* Rust
* ZeroMQ
